# Triangular Meme
[Try it out here!](https://limdingwen.github.io/triangular-meme)

Triangular Meme is a web app that lets you enter two phrases:
```
Homework
Harambe
```

And generates output like this:
```
Homework
Homewor
Homewo
Homew
Home
Hom
Ho
H
Ha
Har
Hara
Haram
Haramb
Harambe
```

Also works with multiple same starting characters:
```
Homework
Homewor
Homewo
Homew
Home
Homes
Homesc
Homesch
Homescho
Homeschoo
Homeschool
Homeschooli
Homeschoolin
Homeschooling
```
